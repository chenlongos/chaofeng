## 当前判断

当前已验证的是 Octos 编排最小闭环，不是轮足真机移动、操作或视觉判定闭环。下一步应该把这次最小 demo 的成功路径抽象成三类接口需求：

- 移动技能接口；
- 操作技能接口；
- 复合任务接口。

在接口稳定前，不应直接把 demo 成功等同于真机编排已完成。
